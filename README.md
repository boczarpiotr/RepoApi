# RepoApi

Run application with below:
>mvn spring-boot:run

Then use this url: 
> http://localhost:8080/getRepos/{name}

to get all of user named {name} information about his/her repositories and commits.

<h3> Tech stack </h3>

- Java 16
- Maven
- Spring-boot
- Lombok
- Jackson
- Gson
